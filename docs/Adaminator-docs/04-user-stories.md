# User Stories

- As an admin I want to create a tournament.
- As an admin I want to add participants.
- As an admin I want random seeding.
- As an admin I want automatic bracket generation.
- As an admin I want to record match results.
- As an admin I want to override match format for specific matches.
- As an admin I want to undo the last result.
- As a viewer I want to see the live bracket.
