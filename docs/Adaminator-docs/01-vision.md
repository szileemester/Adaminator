# Adaminator 🏆

## Vision

**Adaminator** is a lightweight tournament management system focused on fast setup and clean bracket management.

### Goals

- Create tournaments in seconds
- Automatically generate brackets
- Record results with minimal clicks
- Support Single and Double Elimination
- Provide a public read-only view
- Be generic enough for multiple esports

### Target User

A single administrator managing tournaments for friends, LAN parties or small events.

### Tech Stack

- Backend: .NET
- Frontend: React + TypeScript
- Database: PostgreSQL
